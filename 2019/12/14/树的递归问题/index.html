树的递归问题

发表于 2019-12-14 | 分类于 数据结构与算法 ， 树 |
| 字数统计: 1.3k 字 | 阅读时长 ≈ 6 分钟

因为树的性质，许多问题可以用递归求解，本文总结一下树常见的问题的递归方法

Leetcode104.树的深度

Leetcode104.树的深度

树的深度 = 左右子树深度较大值 + 1

Leetcode110.平衡二叉树

Leetcode110.平衡二叉树

平衡二叉树 = 左右子树为平衡二叉树 + 左右子树高度差 <= 1

Leetcode543.二叉树的直径

Leetcode543.二叉树的直径

二叉树直径 = 二叉树任意两个结点之间路径的最大值

Leetcode226.翻转二叉树

Leetcode226.翻转二叉树

翻转二叉树 = 翻转左子树 + 翻转右子树 + 交换左右子树位置

Leetcode617.合并二叉树

Leetcode617.合并二叉树

合并二叉树 = 相加根结点值 + 合并左子树 + 合并右子树

Leetcode112.路径总和

Leetcode112.路径总和

判断是否存在一个路径和的值是给定的值

是否存在这样的路径 = 和减当前节点值 + 左子树是否存在这样的路径/右子树是否存在这样的路径

Leetcode437. 路径总和 III

Leetcode437. 路径总和 III

对于包括根的路径，可以减去根的值之后对左子树和右子树进行递归操作

此题不一定包括根，那么分情况为包括根和不包括根，前者用上面的方法，后者即对左子树和右子树进行递归。

也就是说此题通过两个递归完成。

Leetcode572.另一个树的子树

Leetcode572.另一个树的子树

一个树是否为另一个树的子树 = 遍历所有节点使每个节点成为另一个树根节点，遍历判断是否相同。

Leetcode107.对称二叉树

Leetcode107.对称二叉树

二叉树是否对称 = 左右子树对称

两棵树对称 = 树1左对称树2右 + 树1右对称树2左

Leetcode111.二叉树最小深度

Leetcode111.二叉树最小深度

二叉树最小深度 = 左右子树最小深度的小值 + 1

Leetcode404.左叶子之和

Leetcode404.左叶子之和

对于左子树，要判断是否为叶子节点

对于右子树，直接递归

Leetcode124.二叉树的最大路径和

Leetcode124.二叉树的最大路径和

由下向上DFS求出每个节点的最大贡献

Leetcode687.最长同值路径

Leetcode687.最长同值路径

由下向上DFS求以当前节点为一端的最长同值路径
